# CodersArena

A web app which connect coders and allows to post stuff(like images,,text,links)and other users are
allowed to like , appreciate and collaborate on particular idea to develop it together.
 Built on techstack Backend-Nodejs,Expressjs Frontend-HTML,SCSS,EJS Database-MongoDB.
 Middleware used-connect-flash,express-session,mongoose,multer,passport,passport-google-oauth,
passport-jwt,socket.io,etc.
